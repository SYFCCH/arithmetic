# 树的数据结构  
```java
public TreeNode{
    TreeNode left;
    TreeNode right;
    int val;
    
    public TreeNode() {}
            
    public TreeNode(int val) {this.val = val;}
    
    public TreeNode(int val,TreeNode left,TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }        
    }

```
# 1. 树的前中后序遍历  
```java
class Solution {
    List<Integer> list = new ArrayList<>();
    
    public List<Integer> preorderTraversal(TreeNode root) {
        preorder(root);
        return list;
    }

    public void preorder(TreeNode root) {
        //递归终止条件  
        if(root == null) {
            return;
        }
        list.add(root.val);
        preorder(root.left);
        preorder(root.right);
    }
}



```
